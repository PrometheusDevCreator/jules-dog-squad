# Jules' Dog Squad

Professional dog walking website for South Manchester.

## About

Jules' Dog Squad provides professional dog walking services across Chorlton, Stretford, Sale, Didsbury and surrounding areas of South Manchester. This is a single-page responsive website with an integrated WhatsApp-based booking system.

## Features

- Responsive design optimised for mobile and desktop
- Interactive booking calendar with time slot selection
- WhatsApp integration for bookings and enquiries (no backend required)
- Photo gallery with lightbox viewer
- Service information and pricing
- Testimonial section
- Contact section with WhatsApp and phone links

## Tech Stack

- Single self-contained HTML file (no external dependencies except Google Fonts)
- All imagery embedded as base64 data URIs for zero-dependency hosting
- Pure CSS animations and interactions (no frameworks)
- Vanilla JavaScript for calendar, booking flow and UI

## Deployment

Hosted on Vercel, deployed from this repository. Any push to `main` triggers automatic redeployment.

## Domain

www.julesdogsquad.co.uk

## Booking System

Currently Level 1 (WhatsApp-based):
- User selects date and preferred time from the calendar
- Fills in their details (name, phone, dog, walk type, notes)
- Clicking "Send Booking via WhatsApp" opens a pre-filled WhatsApp message to Jules
- Jules confirms manually

Future upgrade path: Level 3 (Supabase/Firebase backend with admin panel for real-time availability management).
